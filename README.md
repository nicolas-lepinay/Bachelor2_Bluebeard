# Bachelor2_Bluebeard
A bookshop e-commerce website made in PHP8 and MySQL.\
Features: \

◆ Signup, login.\
◆ Cart and checkout.\
◆ Payment with PayPal API.\
◆ Wishlist.\
◆ Account page.\
◆ Search bar.\
◆ Feedback and ratings.\
◆ Admin dashboard.\
